# YouGile Workflow Integration Test

Этот файл создан для проверки работы интеграции GitHub Actions с YouGile.

## Что проверяется

1. **При открытии PR** — задача MF-X перемещается в колонку "Ревью"
2. **При мерже PR** — задача MF-X перемещается в колонку "Готово"

## Как использовать

Укажите ID задачи в названии PR:
- `feat(module): MF-123: описание`
- `fix(api): MF-456: исправление бага`
